# Viral Content Strategy Research - Quick Start Guide

## 📋 Overview

This research aims to develop a systematic content strategy that generates viral videos leading to 10,000 active app users within 3-6 months. The strategy focuses on leveraging BookBridge's unique assets (founder story, testimonials, FREE value prop) through optimized content creation, platform-specific strategies, and conversion optimization.

## 🎯 Research Goal

Create a content strategy that:
- Generates 1,000+ views per video consistently
- Converts 5-10% of viewers into app signups
- Fits a solo creator's schedule (10am-1am, batch creation)
- Uses only free/low-cost tools ($0 budget)
- Is sustainable long-term (avoid burnout)

## 🚀 How to Run This Research

### Step 1: Read the Master Plan
Read `CONTENT_STRATEGY_RESEARCH_PLAN.md` for full context, constraints, and research questions.

### Step 2: Read Context Files
Before running agents, ensure you've reviewed:
- `docs/implementation/ARCHITECTURE_OVERVIEW.md` - App features and capabilities
- `docs/analytics/GOOGLE_ANALYTICS_OVERVIEW.md` - Current user metrics and retention
- User questionnaire responses (provided in conversation) - Current content performance, assets, constraints

### Step 3: Run Each Agent Prompt
Copy the prompt from the master plan for each agent and run it in Cursor/Claude:

1. **Agent 1:** Viral Content Strategy Expert
   - Prompt location: `CONTENT_STRATEGY_RESEARCH_PLAN.md` → Agent 1 section
   - Save to: `Agent1_Viral_Content_Expert_Findings.md`
   - Focus: Hook creation, format optimization, viral content patterns, platform-specific strategies

2. **Agent 2:** Social Media Growth Specialist
   - Prompt location: `CONTENT_STRATEGY_RESEARCH_PLAN.md` → Agent 2 section
   - Save to: `Agent2_Social_Media_Growth_Specialist_Findings.md`
   - Focus: Audience behavior, community building, cross-platform growth, influencer collaboration

3. **Agent 3:** Conversion Optimization Expert
   - Prompt location: `CONTENT_STRATEGY_RESEARCH_PLAN.md` → Agent 3 section
   - Save to: `Agent3_Conversion_Optimization_Expert_Findings.md`
   - Focus: Conversion funnel, CTA optimization, landing page optimization, value proposition

4. **Agent 4:** Systems Thinking Strategist
   - Prompt location: `CONTENT_STRATEGY_RESEARCH_PLAN.md` → Agent 4 section
   - Save to: `Agent4_Systems_Thinking_Strategist_Findings.md`
   - Focus: Content creation workflows, free tools, time-saving techniques, scalable systems

### Step 4: Run Synthesis Agent
After all 4 agents complete their research, run the Synthesis Agent prompt:
- Prompt location: `CONTENT_STRATEGY_RESEARCH_PLAN.md` → Synthesis Agent section
- Save to: `AGENT_SYNTHESIS.md`
- Focus: Combine all findings into 90-day content calendar, execution plan, and resource guide
