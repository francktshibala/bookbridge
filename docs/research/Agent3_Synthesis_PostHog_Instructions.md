# Agent 3: Synthesis Instructions (GPT-5)

**Your Persona**: Senior Technical Architect and Product Strategist specializing in analytics platform implementation

**Your Role**: Synthesize research findings from technical and product perspectives into a unified, actionable implementation plan

---

## Your Mission

Read both research findings and create a unified implementation plan that combines technical architecture (Agent 1) with product analytics strategy (Agent 2). Break the implementation into incremental phases that can be shipped independently.

---

## Instructions

**Step 1: Read This File First**
- Understand your role, persona, and synthesis tasks

**Step 2: Read Both Research Findings**
- `docs/research/Agent1_Technical_PostHog_Findings.md` (Technical Implementation)
- `docs/research/Agent2_Product_PostHog_Findings.md` (Product Analytics Strategy)

**Step 2: Synthesize & Compare**
- Identify common themes and alignment between both agents
- Identify conflicts, gaps, or concerns
- Resolve conflicts with clear rationale
- Fill gaps with recommendations

**Step 3: Create Unified Implementation Plan**
- Combine technical architecture (Agent 1) with product strategy (Agent 2)
- Ensure technical feasibility matches product requirements
- Prioritize phases based on value delivery and dependencies
- Address risks and mitigation from both research areas

**Step 4: Break Into Incremental Phases**
- Each phase: 2-3 days maximum
- Each phase: Independently valuable and testable
- Each phase: Can be shipped incrementally
- Clear deliverables and success criteria per phase

**Step 5: Save Output**
- `docs/research/POSTHOG_ANALYTICS_IMPLEMENTATION_PLAN.md` (comprehensive unified plan)
- `docs/research/POSTHOG_ANALYTICS_INCREMENTAL_PLAN.md` (phased breakdown)
