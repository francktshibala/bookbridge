## 📋 EMOTION CATEGORIES & GRADATIONS

### **Category 1: NEGATIVE EMOTIONS (Fear/Anxiety)**

#### **1. Nervous Series**
- **Level 1:** Worried (mild concern)
- **Level 2:** Anxious (moderate anxiety)
- **Level 3:** Panicked (extreme fear)

**Example Sentences:**
- "I'm worried about my test tomorrow." (A1)
- "She feels anxious before speaking in public." (A2)
- "He panicked when he lost his passport." (B1)

**Situational Use:**
- Worried: Test, deadline, small problem
- Anxious: Job interview, presentation, important event
- Panicked: Emergency, crisis, extreme situation

**Visual Ideas:** Face expressions from calm → tense → terrified

## 🎬 ENHANCED VIDEO STRUCTURE (For Maximum Views)

### **Current Structure (45-90s):**
1. Introduce feeling (10 sec)
2. Gradations (30 sec)
3. Visual demo (20 sec)
4. Shadow practice (20 sec)
5. Situational swap (10 sec)

### **Enhanced Structure (60-90s) - OPTIMIZED FOR VIEWS:**

**0-3 seconds: HOOK (Critical for TikTok)**
- "You're saying 'nervous' wrong" OR
- "POV: You're trying to explain your feelings in English" OR
- "3 ways to say you're scared (you only know 1)"

**3-8 seconds: PROBLEM SETUP**
- "Most English learners only know one word for each feeling"
- "But there are 3 levels: mild, medium, extreme"
- "Today: Nervous"

**8-25 seconds: GRADATIONS (with visuals)**
- "Level 1: Worried" [Show image + text overlay]
- "I'm worried about my test." [Shadow pause 2s]
- "Level 2: Anxious" [Show image + text overlay]
- "She feels anxious before speaking." [Shadow pause 2s]
- "Level 3: Panicked" [Show image + text overlay]
- "He panicked when he lost his passport." [Shadow pause 2s]

**25-40 seconds: VISUAL COMPARISON**
- Show all 3 images side-by-side
- "See the difference?" [2s pause]
- "Worried → Anxious → Panicked" [Text overlay]

**40-55 seconds: SHADOW PRACTICE (3 sentences)**
- "Repeat after me:"
- "I'm worried about my test." [5s pause]

- "He panicked when he lost his passport." [5s pause]

**55-70 seconds: SITUATIONAL USE**
- "Use 'worried' for: tests, deadlines, small problems"
- "Use 'anxious' for: job interviews, presentations"
- "Use 'panicked' for: emergencies, crises"

**70-90 seconds: CTA + CHALLENGE**
- "Can you use all 3 words in one sentence? Comment below!"
- "Follow for Day 2: Scared (3 ways)"
- "Save this video to practice later!"

## 🎨 VISUAL REQUIREMENTS

### **For Each Emotion, You Need:**

1. **3 Emotion Face Images** (one per gradation level)
   - Level 1: Mild expression
   - Level 2: Moderate expression
   - Level 3: Extreme expression
   - Style: Realistic, clear facial expressions, good lighting
   - Format: Square (1:1) for Instagram, vertical (9:16) for TikTok

2. **Text Overlays** (to add in video editing)
   - Bold emotion words (e.g., "WORRIED", "ANXIOUS", "PANICKED")
   - White or yellow text, large font (200% scale)
   - Position: Bottom third of screen

3. **Background** (optional but recommended)
   - Simple, clean background (not distracting)
   - Color gradient matching emotion (blue for sad, red for angry, etc.)

### **Visual Best Practices:**
- ✅ High contrast (emotion clearly visible)
- ✅ Consistent style across all 3 images
- ✅ Professional quality (not blurry)
- ✅ Facial expressions match emotion intensity
- ✅ Diverse representation (different ages, genders, ethnicities)

## 🚀 PRODUCTION WORKFLOW

### **Step 1: Script Writing (5 min per video)**
- Use script template above
- Replace emotion words and sentences
- Add pause markers `[PAUSE Xs]`
- Validate duration (aim for 60-90s)

### **Step 2: Audio Generation (3 min per video)**
- Use ElevenLabs API (Jane voice)
- Generate audio for each text segment
- Add silence segments for pauses
- Concatenate using FFmpeg
- Export: MP3, 64kbps, Mono, 44.1kHz

### **Step 3: Visual Preparation (10 min per video)**
- Generate 3 emotion face images (AI or stock)
- Create text overlays (Canva templates)
- Prepare background (optional)

### **Step 4: Video Assembly (5 min per video)**
- Import audio + visuals into CapCut
- Sync text overlays with audio
- Add closed captions (yellow, all caps, 200% scale)
- Export for TikTok (9:16), Instagram (9:16), YouTube Shorts (9:16)

### **Step 5: Optimization (2 min per video)**
- Add trending hashtags: #learnenglish #speakenglish #esl #emotions #vocabulary
- Write engaging caption with challenge/question
- Schedule posts (best times: 7 AM, 5 PM viewer timezone)
