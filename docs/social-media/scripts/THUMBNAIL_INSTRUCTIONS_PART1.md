# Thumbnail Creation Instructions - "He Arrived with $20 and a Dream" Part 1

## Brand Colors
- **Yellow:** Text color (use for highlights, important numbers, main headlines)
- **White:** Text color (use for secondary text, subheadings)
- **Black:** Background color (solid black or dark background)

## Design Requirements

### Dimensions
- **1280x720px** (YouTube standard)
- **Format:** PNG (high quality)

### Layout
- **Left side (50%):** Emotional image (immigrant at airport, or person looking determined/hopeful)
- **Right side (50%):** Large, bold text overlay

### Text Elements (Choose ONE main headline)

**Option 1 (Curiosity-driven):**
- Main text: "HE ARRIVED WITH $20" (Yellow text, extra large, bold)
- Subtext: "Then he lost EVERYTHING" (White text, large, bold)
- Bottom: "PART 1" (Yellow text, medium)
- Background: Black

**Option 2 (Emotional):**
- Main text: "$20 → MILLIONAIRE → NOTHING" (Yellow text, extra large, bold)
- Subtext: "What happened next?" (White text, large, bold)
- Bottom: "PART 1" (Yellow text, medium)
- Background: Black

**Option 3 (Dramatic):**
- Main text: "ONE PHONE CALL" (Yellow text, extra large, bold)
- Subtext: "Changed Everything" (White text, large, bold)
- Bottom: "PART 1" (Yellow text, medium)
- Background: Black

### Visual Elements
- **Image:** Show person (Diego) looking determined/hopeful OR arriving at airport with suitcase OR restaurant fire scene (dramatic)
- **Contrast:** High contrast between text and background (text must be readable on mobile)
- **Emotion:** Convey hope, struggle, or dramatic tension

### Typography
- **Font:** Bold sans-serif (Impact, Bebas Neue, or similar)
- **Size:** Extra large for main text (readable when thumbnail is small)
- **Style:** All caps for main headline

### Color Usage
- **Yellow text:** Use for "$20", "MILLIONAIRE", "PART 1", main headlines, or key dramatic words
- **White text:** Use for secondary text, subheadings, supporting information
- **Black background:** Solid black or dark background (no other background colors)

### Click-Worthy Elements
- **Numbers:** "$20" prominently displayed (yellow)
- **Emotional words:** "LOST EVERYTHING", "CHANGED EVERYTHING"
- **Curiosity:** "What happened next?" or question format
- **Part number:** "PART 1" clearly visible (creates series expectation)

### Pro Tips
- Text must be readable on mobile (test at small size)
- Use high contrast: Yellow text and white text on black background
- Don't overcrowd - focus on ONE main message
- Emotional image should match story mood (hope → struggle → loss)
- Create curiosity gap (don't reveal the ending)
- Ensure yellow and white text stand out clearly against black background

**Story Context for Image Generation:**
- Immigrant arriving in America with suitcase
- Person working hard (restaurant setting)
- Fire/destruction scene (dramatic moment)
- Person looking determined/hopeful despite struggle
