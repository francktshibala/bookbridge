## 🎨 Visual Style Reference

**Color Scheme:**
- **Yellow:** Key points, methods, positive messages
- **Red:** Lies, wrong methods, problems
- **Green:** Solutions, CTAs, free offers
- **White:** General information, quotes

**Font Sizes:**
- **Extra Large:** Main hooks, key stats (90 DAYS, 65%, 500 WORDS)
- **Large:** Method names, important points
- **Medium:** Supporting text, month labels

**Animations:**
- **Fade in/out:** Standard (0.5s each)
- **Pulse:** For "FREE" offers and CTAs
- **Shake:** For "WRONG" emphasis
- **Glow:** For important stats

3. **Mobile-first:** Test on phone - text must be readable
4. **Contrast:** Ensure text stands out against background
5. **Timing:** Let overlays breathe - don't rush transitions
